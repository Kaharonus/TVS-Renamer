# TVS-Renamer
C# Written app that renames TV Show episodes. At least slight change is released daily.

episodes get renamed to format:

**$ShowName** - S**$SeasonNumer**E**$EpisodeNumber** - **$EpisodeName**

Example:

Futurama - S01E01 - Space Pilot 3000
